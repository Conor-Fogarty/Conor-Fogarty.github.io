My SQL Experience : Data Analysis of Motorcycle Crash Data in Massachusetts

Overview

Utilized advanced SQL skills within the ArcGIS Pro environment to conduct a thorough examination of a detailed CSV dataset obtained from the Massachusetts Department of Transportation (MassDOT). This dataset encompassed 10,709 distinct crash records, each containing an extensive array of data spanning across 79 variables. Enhanced the analysis by integrating geographical information, including town and state details sourced from MassGIS. Transformed the analyzed data into visually engaging reports using Excel, highlighting significant trends and patterns to facilitate comprehension. Delivered a comprehensive presentation seen here, effectively communicating the insights derived from the analysis.
